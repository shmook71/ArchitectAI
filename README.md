# 🚀 ArchitectAI – AI Architecture Evaluation Dashboard

ArchitectAI is a full-stack AI-powered dashboard designed to evaluate system architecture quality based on weighted engineering pillars.

It provides automated scoring, risk classification, maturity level detection, and historical trend tracking.

---

## 🧠 What It Does

ArchitectAI analyzes architecture inputs and generates:

- ✅ Overall Score (0–100)
- ⚠️ Risk Level Classification
- 🏗 Maturity Stage (Basic → Advanced)
- 📊 Trend Analysis Over Time
- 📁 Run History Tracking
